Esta asignatura introduce las herramientas esenciales para el desarrollo con PHP, comenzando con la instalación y configuración del entorno de programación. Los estudiantes aprenderán a instalar PHP 8 y Visual Studio Code como editor principal, además de configurar las extensiones necesarias para un desarrollo eficiente como PHP Server, PHP Debug e Intelephense para obtener inteligencia de código avanzada.

Un componente fundamental del curso será el aprendizaje de Git y GitHub para el control de versiones. Los estudiantes crearán su primera cuenta en GitHub, instalarán GitHub Desktop y gestionarán su primer repositorio, comprendiendo el flujo de trabajo básico con comandos como add, commit, push y pull para mantener un historial detallado de cambios y minimizar riesgos de pérdida de datos.

La asignatura tiene un enfoque práctico donde se combinará la configuración del entorno de desarrollo con las mejores prácticas en control de versiones, sentando las bases para el desarrollo profesional de aplicaciones web con PHP y preparando a los estudiantes para trabajar en entornos de desarrollo colaborativo utilizando herramientas estándar de la industria.
